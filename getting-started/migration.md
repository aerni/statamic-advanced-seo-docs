# Migration

Advanced SEO can help you migrate from SEO Pro and Aardvark SEO. The migrator will process your entries and terms and transform the existing data into the format expected by Advanced SEO.

Run `php please seo:migrate` and pick the migration of your choice:

```bash
php please seo:migrate

Choose your migration:
  [0] Aardvark SEO
  [1] SEO Pro
```

{% hint style="info" %}
Note that the migrator doesn't migrate any config files and settings.
{% endhint %}
