# Editions

Advanced SEO is available in two editions. The **Free** edition covers everything you need for solid on-page SEO. The **Pro** edition unlocks advanced features like sitemaps, multi-site support, and AI content generation.

## Free

Everything you need for solid on-page SEO. Compose titles and descriptions using dynamic tokens pulled from your content fields, preview how they look in search results and social media, and reset any value back to its inherited default.

* Intuitive editing with dynamic token inputs and reset-to-default
* Live-updating search and social previews
* Complete meta tag output (Open Graph, hreflang, canonical, JSON-LD, and more)
* Global site defaults (site name, trackers, verification, and more)
* Collection and taxonomy defaults inherited by entries and terms
* Git integration
* Migration from SEO Pro and Aardvark SEO

## Pro

Unlock the full potential of Advanced SEO. Includes everything from the Free edition, plus:

* Sitemaps
* Multi-site support
* AI content generation
* Social images generator
* Granular permissions
* SEO for custom routes
* Custom tokens for title and description inputs
* GraphQL API
* Eloquent driver
