# Customizing Fields

{% hint style="warning" %}
Blueprint customization has been removed in v3. If you previously customized the Advanced SEO fieldsets or used the `advanced_seo` fieldtype in your blueprints, please refer to the [upgrade guide](../getting-started/upgrade.md#blueprint-customization-removed) for migration steps.
{% endhint %}

The fields shown in the SEO tab of your entries and terms are managed entirely by the addon. You can disable features in the `config/advanced-seo.php` config file to remove their corresponding fields from the SEO tab.
