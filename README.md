# Introduction

Advanced SEO is a Statamic addon that gives you full control over your site's SEO, from meta tags and social sharing to sitemaps and structured data. It's designed to handle the technical details automatically while giving content editors a simple, intuitive interface to work with.

Advanced SEO is available in two [editions](getting-started/editions.md). The **Free** edition covers essential on-page SEO with a polished editing experience. The **Pro** edition unlocks advanced features like sitemaps, multi-site support, AI content generation, and more.

## Features

* Dynamic token inputs with reset-to-default
* Live-updating search and social previews
* Complete meta tag output (Open Graph, hreflang, canonical, JSON-LD, and more)
* Global site defaults (site name, trackers, verification, and more)
* Collection and taxonomy defaults inherited by entries and terms
* Git integration
* Migration from SEO Pro and Aardvark SEO
* Sitemaps (Pro)
* Multi-site support (Pro)
* AI content generation (Pro)
* Social images generator (Pro)
* Granular permissions (Pro)
* SEO for custom routes (Pro)
* Custom tokens for title and description inputs (Pro)
* GraphQL API (Pro)
* Eloquent driver (Pro)

## Requirements

* PHP 8.4+
* Laravel 12+
* Statamic 6+
